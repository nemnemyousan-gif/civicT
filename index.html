シビックテックの取り組み

Civic Tech Community Open Data

私は、マンション管理や地域活動の現場で見えてくる課題を、デジタルとデザインの力で少しずつ改善していく取り組みを続けています。 派手なプロジェクトではなく、生活に密着した「小さな困りごと」を解決することを大切にしています。

  • マンション管理のデジタル化・情報整理
    管理規約・総会資料・修繕履歴などを体系化し、住民が迷わずアクセスできる構造を設計。 紙中心の運用からクラウド管理への移行を検討し、情報アーキテクチャの観点から改善案を作成しています。
  • 多言語対応（英語化）によるアクセシビリティ向上
    外国籍住民向けに重要文書の英語版を作成し、「翻訳」ではなく「伝わる文書」を意識した表現を研究。 GitHub 上で翻訳プロセスを管理し、変更履歴を残す仕組みを試行しています。
  • 地域イベント（マンション内祭り）の運営改善
    主催者の高齢化により運営が限界に近づいている問題を調査。 参加者管理・備品リスト・スケジュールなどをデジタル化し、属人化を減らす仕組みを提案。 「誰でも引き継げる運営マニュアル」を作るため、ワークフローの可視化を行っています。
  • 住民アンケートの設計とデータ分析
    修繕計画やイベント運営に関するアンケートを設計し、回答データを可視化。 合意形成をスムーズにするための「住民の声の見える化」を実践しています。
  • GitHub を使ったオープンな学習・共有
    学習メモや文書テンプレート、改善案を GitHub で公開。 試行錯誤のプロセスごとオープンにすることで、同じ課題を持つ人の参考になる形を目指しています。

小さく作り、試し、改善し、その過程を共有すること。 その積み重ねこそが、私の考えるシビックテックです。
